Milu

One window for everything you keep open.

麋鹿 又叫四不像

The Père David's deer. In Chinese it's also called sì bù xiàng — "not like any" — because it has the antlers of a deer, the face of a horse, the hooves of a cow, and the tail of a donkey. It looks like everything, and resembles nothing.

That's the idea. Milu is a markdown editor, a code editor, a folder browser, a terminal, a real browser, a git client, an AI agent host. It looks like all of them, and isn't quite any of them.

One window. Recursive split panes. tmux-style sessions.

  • Markdown, code, CSV, JSON, diff, PDF
  • Folders, embedded browser, terminal
  • Git client with inline diffs & AI review
  • ACP-native — host Claude Code, Gemini CLI
Download for Apple Silicon

macOS 12+ · Intel build · GitHub

First launch shows "Milu is damaged"?

Milu isn't notarized with Apple, so macOS quarantines the download. In Terminal, run: 

xattr -dr com.apple.quarantine /Applications/Milu.app

Re-run this after every new download — fresh downloads get re-quarantined.
